Let $K$ be a closed, bounded, convex set in some Banach space and let $T:K\rightarrow K$ be a map which satisfies $\|T(x)-T(y)\|\leq \|x-y\|$ for all $x,y\in K$. Prove that for every $\ve>0$ there exists a point $z \in K$ such that $\|T(z)-z\| < \ve$.

Perturb $T$ to a nearby uniform contraction.

Fix a base point $p\in K$. For any $0 < c < 1$, define a new map $T_c:K\rightarrow K$ by $T_c(x):=(1-c)p+cT(x)$. We have $\|T_c(x)-T_c(y)\|=c\|T(x)-T(y)\| \leq c\|x-y\|$, so by the contraction mapping principle $T_c$ has a (unique) fixed point $z_c \in K$. Now $$ \|T(z_c)-z_c\|= \Big( \frac{1}{c}-1 \Big) \| z_c-p \| \leq \Big( \frac{1}{c}-1 \Big) \diam(K) \leq \ve $$ if $c$ is close enough to 1.

Remark. If $K$ is compact, it follows that $T$ has a fixed point in $K$. But this is not generally the case. For example, let $K$ be the closed unit ball in the Banach space of all sequences $x= \{ x_n \}$ of (real or complex) numbers with $\lim_{n \to \infty} x_n = 0$, equipped with the norm $\| x \| =\sup_n |x_n|$. Define $T(\{ x_n \})= \{ y_n \}$ by $y_1:=(1+\|x\|)/2$ and $y_n:=(1-2^{-n})x_{n-1}$ for $n\geq 2$. It is easy to see that $T$ maps $K$ to itself and satisfies $\|T(x)-T(y)\| < \|x-y\|$. However, $T$ has no fixed points since $T(x)=x$ implies $x_n > \prod_{i=1}^n (1-2^{-i})$, contradicting $x_n \to 0$.

Along the same lines, if $K$ is a compact and convex subset of a Banach space $X$, and if $T: X \to X$ is a continuous linear map which sends $K$ to itself, then $T$ has a fixed point in $K$. To see this, choose $x\in K$ and define the sequence $\{ y_n \}$ in $K$ by $y_n:=(1/n) \sum_{i=0}^{n-1} T^{\circ i}(x)$. By compactness, some subsequence of $\{ y_n \}$ converges to a point $y\in K$. Since $Ty_n -y_n=(T^n x -x)/n$, it follows that $Ty=y$.
