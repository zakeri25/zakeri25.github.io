A binary vector in $\RR^n$ is a vector whose coordinates in the standard basis consist only of $0$'s and $1$'s. Find the maximum number of binary vectors in an $m$-dimensional subspace $V$ of $\RR^n$.

The answer is $2^m$. Think of projecting $V$ onto various $m$-dimensional coordinate planes of $\RR^n$.

Let $\{ e_1, \ldots, e_n \}$ denote the standard basis of $\RR^n$. For each set $S \subset \{ 1, \ldots, n \}$ of size $m$ consider the linear span $W_S$ of $\{ e_i \}_{i \in S}$ which is an $m$-dimensional subspace of $\RR^n$, and define the projection $\Pi_S:\RR^n \to W_S$ by $\Pi_S(\sum_{i=1}^n a_i \, e_i)=\sum_{i \in S} a_i \, e_i$. Since $\dim(V)=m$, we can find an $S$ for which $\Pi_S$ restricts to an isomorphism $V \to W_S$. (Here is why: Take an $m \times n$ matrix in reduced row echelon form whose row space is $V$. The positions of the columns containing the leading $1$'s in this matrix gives such a set $S$.) Note that $\Pi_S$ sends every binary vector in $\RR^n$ to a binary vector in $W_S$. Since $\Pi_S$ is injective on $V$, the number of binary vectors in $V$ is at most equal to the number of those in $W_S$, namely $2^m$.
