We choose $n$ numbers in $[0,1]$ independently with uniform distribution. What is the expected value of their minimum?

The answer is $1/(n+1)$.

Let $x_1, \ldots, x_n$ denote the chosen numbers. Consider the set of choices were $x_1 \leq x_2 \leq \cdots \leq x_{n-1} \leq x_n$. The contribution of these choices to the expected value of the minimum is $$ \int_0^1 \int_0^{x_n} \cdots \int_0^{x_2} x_1 dx_1 \cdots dx_{n-1} \, dx_n = \frac{1}{(n+1)!}. $$ The set of all choices is partitioned into $n!$ such sets obtained by permuting the variables and they all give the same contribution by symmetry. Hence the expected value of the minimum is $n!/(n+1)!=1/(n+1)$.
