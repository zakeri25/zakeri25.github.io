Let $V$ be a finite-dimensional vector space over $\CC$ and $T:V \to V$ be a linear map with the kernel and range $N_T$ and $R_T$, respectively. Prove that $V=N_T \oplus R_T$ if and only if $x^2$ does not divide the minimal polynomial of $T$.

The direct sum condition is equivalent to $N_T \cap R_T = \{ 0 \}$. Look at the constant and linear terms of the minimal polynomial of $T$.

Since $N_T$ and $R_T$ are subspaces of $V$ with complementary dimensions, it is clear that $V = N_T \oplus R_T$ if and only if $N_T \cap R_T = \{ 0 \}$. Let $p(x)=x^n+a_{n-1}x^{n-1}+\cdots+a_2x^2+a_1x+a_0$ be the minimal polynomial of $T$. First suppose $x^2$ does not divide $p(x)$. If $a_0 \neq 0$, then $T$ is invertible and $N_T \cap R_T= \{ 0 \}$ trivially holds. If $a_0=0$ but $a_1 \neq 0$, take any $w \in N_T \cap R_T$ and write $w=T(v)$ for some $v \in V$. The relation $T^n(v)+\cdots +a_1T(v)=0$ then gives $a_1w=0$, so $w=0$.

Conversely, suppose $x^2$ divides $p(x)$ so $a_0=a_1=0$. The polynomial $x^{n-1}+a_{n-1}x^{n-2}+\cdots+a_2x$ does not annihilate $T$, so there is a $v \in V$ such that $w:=T^{n-1}(v)+a_{n-1}T^{n-2}(v)+\cdots+a_2T(v) \neq 0$. But $T(w)=p(T)(v)=0$, so $w \in N_T \cap R_T$. This shows $N_T \cap R_T \neq \{ 0 \}$.
