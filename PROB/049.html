Consider a countable dense set $\{ x_n : n \geq 1 \}$ of distinct points in the interval $(0,1) \subset \RR$. For each $n \geq 1$ the point $x_n$ falls into a unique connected component of $(0,1) \sm \{ x_i : i < n \}$ and cuts it into two subintervals of lengths $a_n, b_n$. Prove that $\sum_{n=1}^{\infty} a_n b_n (a_n+b_n)=1/3$.

Write the assertion as $\sum_{n=1}^{\infty} ((a_n+b_n)^3-(a_n^3+b_n^3))=1$. Use the fact that each $a_n+b_n$ equals $a_m$ or $b_m$ for some $m < n$.

The desired equality can be written as $\sum_{n=1}^{\infty} ((a_n+b_n)^3-(a_n^3+b_n^3))=1$. From the construction it is clear that each term $a_n+b_n$ equals $a_m$ or $b_m$ for some $m < n$. Fix $\ve>0$ and use density of $\{ x_n : n \geq 1 \}$ to find a large enough $N>1$ such that the points $x_1, \ldots, x_N$ chop up the inteval $(0,1)$ into pieces of length at most $\ve$. Set \begin{align*} A_N & = \{ m \in [1,N] : a_m \neq a_n+b_n \ \ \text{for all} \ \ m < n \leq N \}, \\ B_N & = \{ m \in [1,N] : b_m \, \neq a_n+b_n \ \ \text{for all} \ \ m < n \leq N \}. \end{align*} Then $$ \sum_{n=1}^N ((a_n+b_n)^3-(a_n^3+b_n^3))=(\underbrace{a_1+b_1}_1)^3-\sum_{n\in A_N} a_n^3 -\sum_{n\in B_N} b_n^3. $$ But $\sum_{n\in A_N} a_n +\sum_{n\in B_N} b_n=1$ and $a_n \leq \ve$ for all $n \in A_N$ and $b_n \leq \ve$ for all $n \in B_N$. It follows that $\sum_{n\in A_N} a_n^3 +\sum_{n\in B_N} b_n^3 \leq \ve^2$, which shows $$ 0 < 1-\sum_{n=1}^N ((a_n+b_n)^3-(a_n^3+b_n^3)) \leq \ve^2. $$
